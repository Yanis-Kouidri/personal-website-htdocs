Qu'est ce que ce site ?

Ce site est le résultat d'un projet étudiant, créé en décembre 2020, il a pour but d'évaluer différents types de produits que l'on peut utiliser dans notre quotidien. Pour l'instant seul l'auteur de ce site peut évaluer un produit mais bientôt tous les visiteurs seront libres de laisser une évaluation sur les produits déjà présents.

Qui est derrière ce site ?

Derrière ce site se trouve Yanis Kouidri, un étudiant en Réseaux et Télécommunications à l'IUT de Sophia-Antipolis.

Quelle est la valeur ajoutée de ce site ?

Ce site permet à chacun de découvrir de nouveaux produits ainsi que de se faire un avis. Les utilisateurs pourront laisser une évaluation sur le site sans aucune censure à partir du moment où l'évalutation respecte les règles fixées par le site. Ce site n'a de lien, rémunéré ou non, avec aucune entreprise, organisation, état, gouvernement, corporation etc... Le site est complétement indépendant et est a but non lucratif. En aucun cas le site ne modifira/supprimera/rajoutera un article/une évaluation sous demande d'un tiers, même si celui-ci propose des fonds.
